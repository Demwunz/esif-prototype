Capital Projects Documents

The following accompanying documents should be provided for all Capital Projects applying for ERDF.

Accompanying documents for Capital Projects applying for ERDF

Evidence that the applicant has/will have control of the site to deliver the project.

Specification of the document or acceptable alternative

Freehold or leasehold title for the project or signed Heads of Terms between applicant and vendor for land/building acquisition.

Advise whether a document is not applicable for the project

Accompanying documents for Capital Projects applying for ERDF

State Aid Report

Specification of the document or acceptable alternative

A State Aid Report, addressed to the Applicant and produced by a suitably qaulified professional organisation (e.g a law firm or accountant) which

  • lists all the Project costs used to determine the State Aid intervention rate
  • gives an opinion as to whether each cost is eligible
  • analyses whether the proposed award to the Applicant meets all the requirements of the specified State Aid scheme
  • sets out all the information required for Article 6 of regulation 651/2014 (if applicable)

If the Applicant intends to use the value of any land or buildings in the match funding calculation, the report must provide an express explanation as to how this is State Aid compliant, if necessary using the information set out in the independent valuation report.
